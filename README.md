# Simplified Ai



## Features
```
----- Main Page -----
1. Project Creation 
2. Data Ingestion (Aws,Gcp,Azure,Local)
3. Project Report 
4. Data Export 

----- ML Life Cycle ------
1. Exploratory Data Analysis API
2. Data Preprocessing API
3. Feature Engineering API
4. Model Training API
   I)  Auto-Matic 
   II) Custom
5. Prediction API
6. Process Scheduling API
7. System Logs
8. History Tracker
```

## Tech Stack
```
Front End: 
1. Html
2. Css
3. Javascript
4. Jquery

Back End:
1. Python
```
